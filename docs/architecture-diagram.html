VAD Decision Logic 
flowchart TD
    Start["Audio Sample 4096 frames"] --> RMS["Calculate RMS Energy"]
    RMS --> Compare{"RMS > 0.02?"}
    Compare -->|"YES"| Speech["Speech Detected"]
    Compare -->|"NO"| Silence["Silence Detected"]
    Speech --> IsSpeaking{"Already speaking?"}
    IsSpeaking -->|"NO"| StartSpeech["Start Speech Timer Reset Silence Timer"]
    IsSpeaking -->|"YES"| Buffer1["Add to Buffer Reset Silence Timer"]
    Silence --> WasSpeaking{"Was speaking?"}
    WasSpeaking -->|"NO"| Ignore["Ignore waiting for speech"]
    WasSpeaking -->|"YES"| CheckDuration{"Silence > 400ms AND Speech > 500ms?"}
    CheckDuration -->|"NO"| Buffer2["Add to Buffer short pause"]
    CheckDuration -->|"YES"| Send["Send Chunk Reset Buffers"]
Challenge Scoring Algorithm (Game Mode) 
flowchart LR
    Input["User Transcription"] --> Norm1["Normalize Text"]
    Challenge["Challenge Phrase"] --> Norm2["Normalize Text"]
    Norm1 --> Lev["Calculate Levenshtein Distance"]
    Norm2 --> Lev
    Lev --> Calc["Similarity = length - distance / length"]
    Calc --> Score["Accuracy % = Similarity x 100"]
    Score --> Color{"Score?"}
    Color -->|">= 90%"| Green["Success + Prize"]
    Color -->|"70-89%"| Blue["Good"]
    Color -->|"< 70%"| Orange["Needs Improvement"]

Formula: similarity = (longer.length - editDistance) / longer.length

Implementácia: Client-side JavaScript v src/templates/index.html
